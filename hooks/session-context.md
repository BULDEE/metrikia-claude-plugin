# Metrikia Plugin Active

You have access to the Metrikia MCP server for ad tracking, ROI analysis, and CRM correlation.

## Skills

| Skill | When to use |
|-------|-------------|
| `/metrikia:weekly-report` | User asks for a weekly performance summary or regular reporting |
| `/metrikia:campaign-audit` | User wants to deep-dive into a specific campaign |
| `/metrikia:lead-pipeline` | User asks about CRM pipeline health, lead quality, or conversion rates |
| `/metrikia:budget-optimizer` | User wants to reallocate budget or optimize spend |
| `/metrikia:creative-analysis` | User asks about creative performance, fatigue, or scaling |
| `/metrikia:attribution-deep-dive` | User wants multi-touch attribution analysis or channel valuation |

## Agents

| Agent | Focus |
|-------|-------|
| `media-buyer` | Campaign optimization, budget allocation, scaling strategies, cross-platform synergy |
| `growth-analyst` | Full-funnel analysis, attribution modeling, revenue correlation, lead quality |

## MCP Tools (17)

**Campaigns:** `list_campaigns`, `get_campaign_performance`, `get_creative_report`, `compare_performance`

**Leads/CRM:** `list_leads`, `get_lead`, `list_deals`

**Attribution:** `get_attribution_journey`, `get_budget_advice`

**Metrics:** `get_metrics`, `get_anomalies`, `get_sync_status`

**Diana AI:** `ask_diana`

**Write:** `create_lead`, `transition_lead`, `create_deal`, `trigger_sync`

## Reminder

Always fetch data with MCP tools before making recommendations. Never guess metrics.
